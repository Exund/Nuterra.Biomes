Generation
BiomeMap handles biome map generation and then heightmap generation
BiomeGroupDatabase is used to get biomes

MapGenerator Layers 
	p = generated value (point)
	Function: initial value of the layer (l)
	Operations: Operation applied to the value
	Apply Operation: operation applied to "p" with "l" as an additionnal parameter 


	Layers algo
	let "p" be the generated point
	p = 0
	for each layer in generator
	  let "l" be the layer value
	  l = layer.function()
	  for each operation in layer
		l = operation(l, variable_buffer)
	  p = layer.apply_operation(p, l)
	clamp p between -1 and 1
	return p

VS Code JSON Schemas config
In VS Code go to File > Preferences > Settings 
Then to Extensions > JSON
Then search for "Schemas" and click "Edit in settings.json"
Then add this at the end: 

"json.schemas": [
    {
        "fileMatch": [
            "*.biome.json"
        ],
        "url": "https://raw.githubusercontent.com/Exund/Nuterra.Biomes/master/SCHEMAS/Biome.schema.json"
    },
    {
        "fileMatch": [
            "*.generator.json"
        ],
        "url": "https://raw.githubusercontent.com/Exund/Nuterra.Biomes/master/SCHEMAS/MapGenerator.schema.json"
